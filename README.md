# Student_Management_System

## Overview

This project was developed for the Advanced Programming Concepts module.

The project demonstrates the implementation of a Student Management System using two different approaches:

- Version A – Traditional Programming
- Version B – AI-Assisted Programming

## Features

- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Display Total Students

## Technologies Used

- Java
- Object-Oriented Programming
- ArrayList
- Exception Handling
- Git
- GitHub
